Code Projects

I enjoy deepening my understanding of physics and machine learning by creating code-driven projects. Below is a selection of these explorations, from my early solar cell simulations to more recent self-driving car experiments.

Simulation tool for solar cell research

These tools were developed during my time as a solar cell researcher (pre-2020). They are no longer under active development, but still serve as helpful references for understanding and analyzing solar cell performance.

Pypvcell

A solar cell simulation package and toolkit for day-to-day solar cell data analysis.

Github Repo

Documentation

Pypvcircuit

A 2D equivalent circuit simulation model for solar cells. I developed a new algorithm that boosts simulation speed by over two-fold.
