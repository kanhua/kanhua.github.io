Traffic light detection (a homework solution)

六 03 3月 2018
By Kan-Hua Lee

Disclaimer: This solution is "overfitted" to the image scenarios in the test environments provided by Udacity. This will not be a solution for more general traffic light detection. "Homework solution" in the title means that it is a solution to a simplifed problem, and the main purpose is more for the student's own enrichment rather than contributing anything useful to the world.

Feature of this implementation

  • It does not use machine learning approach. The model learns from domain-specific knowledge, not from data.

  • It is fast.

Drawback of this model

  • It works reasonable in the test environments of Udacity self-driving car project, but may not be robust enough for other applications.
